# WeatherApp

Simple One page Application which shows the Current weather and the forecast based on the browser's Geolocation.
- Backend developed using Nodejs and Koa
- Frontend developed using React,react-bootstrap
- Solution is containerized 
[![N|Solid](https://github.com/smitha-2020/weatherApp/tree/main/Images/weatherApp.png)](https://github.com/smitha-2020/weatherApp/tree/main/Images/weatherApp.png)
- Solution is deployed in AWS and is available at http://51.20.31.39:3000/

#Steps to replicate:
- clone the Repository
- docker-compose up -d
- Access the React frontpage using http://localhost:3000/
